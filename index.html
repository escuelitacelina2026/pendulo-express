FUNCIONAMIENTO DEL PÉNDULO: LOS MOVIMIENTOS

El péndulo se comunica mediante un código de lenguaje entre la mente consciente e inconsciente. Los movimientos trazan dibujos que el hemisferio derecho entiende y traduce para el hemisferio izquierdo.

Giros (Horario/Anti-horario)

Representan la búsqueda y respuesta de energía circular.

Líneas (Vertical/Horizontal)

Representan la dirección y el flujo de la energía lineal.

DESCUBRE TU CÓDIGO (SI / NO)

Para establecer tu comunicación personal, utiliza estas preguntas de control:

Para obtener un "SI":

Pregunta algo obvio: "¿La nieve es de color blanco?". El movimiento resultante será tu código para el SI. Di en voz alta: "Este movimiento significa SI".

Para obtener un "NO":

Pregunta algo falso: "¿La nieve es de color negro?". El movimiento resultante será tu código para el NO. Di en voz alta: "Este movimiento significa NO".
