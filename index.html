FUNCIONAMIENTO DEL PÉNDULO: LOS MOVIMIENTOS

El péndulo se comunica mediante un código de lenguaje; un código entre la mente consciente e inconsciente, entre el hemisferio cerebral derecho y el cerebral izquierdo.

Hemisferio Izquierdo: Deductivo, lógico, analítico, consciente, exterior, yang, maneja la palabra.
Hemisferio Derecho: Intuitivo, espiritual, holístico, inconsciente, interior, yin, maneja los dibujos.

"La mente domina la materia."

TRAZOS Y DIBUJOS

Los movimientos del péndulo trazan dibujos que el hemisferio derecho entiende y traduce a palabras.

Giros: Sentido Horario y Anti-horario

Líneas: Vertical, Horizontal y Diagonal

DESCUBRE TU CÓDIGO (SI / NO)

Di o piensa: “Solo quiero ser sensible a las radiaciones de mi péndulo para descubrir el código de lenguaje que mantiene conmigo”.

Obtener un SI:

Formula una pregunta obvia. Ej: ¿La nieve es blanca?

El movimiento resultante es tu SI. Di: "Este movimiento significa SI".

Obtener un NO:

Formula una pregunta falsa. Ej: ¿La nieve es negra?

El movimiento resultante es tu NO. Di: "Este movimiento significa NO".

Nota: Si no hay movimiento, existe un bloqueo. Es necesario tener una actitud neutral para que el péndulo se mueva correctamente.
